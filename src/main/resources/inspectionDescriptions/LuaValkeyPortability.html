Reports Valkey-only API used in scripts whose project runtime target is Redis. Valkey adds a server.* alias namespace (server.call, server.pcall, …) and the SERVER_NAME, SERVER_VERSION, and SERVER_VERSION_NUM globals. These are not available on Redis, so a script using them is not portable.

For a server.<member> access a quick fix Replace 'server' with 'redis' rewrites the base to the compatible redis.* namespace, which runs on both servers. The SERVER_* globals have no 1:1 Redis equivalent, so no quick fix is offered.

The inspection is target-aware: it stays silent under a Valkey target (where this API is legal) and under any non-Redis/Valkey target. Switch the project runtime target to Valkey to allow these APIs.
