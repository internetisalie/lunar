Reports problems found by LuaCheck, the external Lua static analyzer, directly in the editor. Each LuaCheck warning or error (for example an unused variable or an undefined global) is shown inline at its source range.

Disabling this inspection turns off the on-the-fly LuaCheck highlighting: the paired external annotator is skipped whenever this entry is unchecked in the current profile.

Configure the LuaCheck executable and command-line arguments under Settings | Tools | LuaCheck.
