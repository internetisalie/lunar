# COMP-02-05: Iconography

As a Lua developer, I want to see distinct icons for different symbol types (locals, parameters, globals) in the completion list.

## Acceptance Criteria
- [x] Display variable icons for local variables.
- [x] Display parameter icons for function parameters.
- [x] Display function icons for global functions/variables.
