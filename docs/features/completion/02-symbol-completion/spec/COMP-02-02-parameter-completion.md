# COMP-02-02: Parameter Completion

As a Lua developer, I want to see function parameters in the completion list so that I can easily use them within the function body.

## Acceptance Criteria
- [x] Suggest parameters of the containing function.
- [x] Support for multiple parameters in a list.
- [x] Support for `self` in method-like functions.
